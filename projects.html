projects

IEEE BAU student branch website

A full website built for the IEEE BAU Student Branch in my university to showcase events, activities, and membership opportunities. It features responsive design, organized content sections — improving visibility and engagement for the branch.

raven

An AI-powered motion recognition project developed for the MotionSense AI competition. The system uses sensor data from smartphones (accelerometer, gyroscope, magnetometer) to classify different motion patterns like walking, running, or standing, and the interface is a Flutter mobile app with a websocket API built using Django for real-time analysis.

minesweeper robot

An intelligent landmine detection robot designed to identify and map buried mines using camera vision and metal detection sensors. I worked in the software and AI development, implementing algorithms for autonomous navigation, object detection, and real-time mapping. The project represented Jordan in the Minesweeper Robot Competition held in Egypt, It later won Third Place in the Jordan Design and Development Bureau’s (JODDB) Creative Excellence Initiative. As part of the award, we developed an advanced second version funded by JODDB, which was showcased at SOFEX 2024, integrating improved AI perception and control systems for enhanced performance in real-world environments.

underwater drone

An underwater drone developed as part of the Orange Summer Challenge, where participants built technology-driven startup ideas. Our project aimed to tackle marine safety and ecosystem balance by integrating AI-based dangerous object detection, specifically identifying lionfish species. The drone is powered by a Raspberry Pi and Pixhawk controller, featuring brushless motors, LED lighting, and a live video feed system. it enables real-time underwater monitoring, making it suitable for environmental research, oceanography, and future commercial applications in marine exploration.

lavamart

An e-commerce platform built using flutter to deliver a smooth and modern shopping experience, with features for browsing, managing carts, and secure checkout. Beyond traditional e-commerce functions, LavaMart integrates an AI-powered chatbot based on the OpenAI API, providing users with intelligent product recommendations, instant support, and conversational shopping assistance.

lavamart API

A custom RESTful API built using ASP.NET Core, designed to power the LavaMart e-commerce platform. It handles key backend operations such as user authentication, product management, and order processing with secure, scalable architecture. The API was structured for smooth integration with both the LavaMart frontend and potential third-party services.

firefighter robot

A rescue-focused robotic system built to operate in simulated fire emergency environments. The robot was engineered to navigate hazardous zones, and assist in rescue missions — combining mechanical precision with reliable sensor-based control. I contributed to the programming and control systems, After months of teamwork filled with long nights, breakthroughs, and tough debugging sessions, our effort was rewarded when the robot won Third Place in the Mechanical Design category at the Firefighting Competition organized by Princess Sumaya University for Technology (PSUT).

sumo robot

An autonomous combat robot designed for Sumo-style robotics competitions, where two robots face off to push each other out of the ring. The robot combines sensor-based opponent detection with fast motor control and strategic movement algorithms for competitive performance. We participated in the competition three times, improving the robot’s design and performance with each version — from refining the mechanical stability and grip to optimizing control logic and decision-making speed. Every iteration brought new lessons in electronics, coding, and teamwork, steadily evolving the robot into a more capable and adaptive contender.
